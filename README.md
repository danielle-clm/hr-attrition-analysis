# HR Employee Attrition Analysis
### Excel data analysis project, IBM HR Dataset

## Overview
This project analyzes employee attrition data from a real-world IBM HR dataset using Microsoft Excel.
The goal was to explore why employees leave a company by building summary tables, practicing core Excel formulas, and visualizing key findings through charts.

---

## Dataset
- **Source:** IBM HR Analytics Employee Attrition Dataset
- **Records:** 1,470 employees — 35 columns
- **Target column:** Attrition (Yes / No)
- **Missing values:** None

---

## Tools Used
- Microsoft Excel

---

## Key Findings

**1. Which department has the highest turnover?**
Sales leads at 21%, followed by HR at 19% and R&D at 14%.

**2. Do overtime employees leave more?**
Yes — 31% of overtime employees left vs only 10% of non-overtime employees.

**3. Does salary affect attrition?**
Yes — Sales Representatives earning $2,626/month have 40% attrition while Managers earning $17,181 have only 5%.

**4. Which age group quits most?**
Employees aged 18–25 leave at a 36% rate, dropping sharply after age 35.

---

## Conclusion
The three main attrition drivers are low salary, working overtime, and young age.
Retention efforts should focus on entry-level, low-salary, overtime-heavy roles.
